## 1. Что выведет код?

```js
console.log(typeof null);
```

A) "object"
B) "null"
C) "undefined"
D) "function"

Ответ: **А**

```js
console.log(typeof null); // "object"
```

Объяснение: Этот баг в JavaScript с 1995 года, так и остался по сегодняшний день

## 2. Что произойдет?

```js
const arr = [1, 2, 3];

arr[10] = 5;
console.log(arr.length);
```

A) 3
B) 4
C) 11
D) 10

Ответ: **C**

```js
const arr = [1, 2, 3];

arr[10] = 5;
console.log(arr.length); // 11

// Index: 1 Value: 2
// Index: 2 Value: 3
// Index: 10 Value: 5

console.log(arr);

// [1, 2, 3, empty~6, 5];
```
